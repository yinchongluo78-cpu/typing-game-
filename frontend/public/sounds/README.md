# 音效文件说明

请将下载的音效文件放到此目录，并按以下命名：

| 文件名 | 用途 | 建议时长 |
|--------|------|---------|
| `key-press.mp3` | 正常打字按键音 | 50-100ms |
| `key-error.mp3` | 打错字提示音 | 100-200ms |
| `sentence-complete.mp3` | 完成一句提示音 | 300-500ms |
| `chapter-complete.mp3` | 完成章节庆祝音 | 1-2s |

## 推荐下载网站

1. **Pixabay** - https://pixabay.com/sound-effects/
2. **Freesound** - https://freesound.org/
3. **Mixkit** - https://mixkit.co/free-sound-effects/

## 搜索关键词

- 打字音效: keyboard click, keystroke, typing
- 错误音效: error beep, wrong, buzz
- 完成音效: ding, notification, pop, success
- 庆祝音效: achievement, level up, victory, fanfare
